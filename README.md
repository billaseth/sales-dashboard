# 📊 Sales & Revenue Analysis Dashboard

An interactive web-based dashboard built using **Python**, **Streamlit**, and **Plotly** to analyze sales performance, track business KPIs, and visualize revenue trends.

---

## 🚀 Key Features
* **File Upload Support:** Upload custom CSV or Excel datasets, or use the built-in sample data.
* **Interactive KPI Metrics:** Real-time tracking of Total Revenue, Total Profit, Units Sold, and Average Order Value (AOV).
* **Dynamic Visualizations:**
  * Monthly Revenue Trends (Line Chart)
  * Top-Performing Products (Bar Chart)
  * Regional Sales Distribution (Pie Chart)
* **Custom Filters:** Filter data dynamically by Region and Product categories using sidebar controls.

---

## 🛠️ Tech Stack
* **Python** (Core Programming)
* **Streamlit** (Dashboard UI Framework)
* **Pandas & NumPy** (Data Processing & Aggregation)
* **Plotly** (Interactive Data Visualizations)
